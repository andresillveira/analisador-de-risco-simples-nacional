# Analisador de Risco - Simples Nacional

Este projeto é uma ferramenta para análise de conformidade e auditoria fiscal de empresas optantes pelo Simples Nacional. A aplicação executa análises com base no Art. 29 da Lei Complementar nº 123/2006, avaliando limites de despesas (Inciso IX) e compras (Inciso X).

## Como Executar Localmente

**Pré-requisitos:** Node.js instalado.

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Configure suas chaves ou variáveis de ambiente no arquivo `.env` (use o [.env.example](file:///d:/analisador-de-risco-simples-nacional/.env.example) como modelo).
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
